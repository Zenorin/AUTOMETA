# Evidence Pack

## Use when
- The task matches this skill's description.
- The change may affect durable behavior, tests, docs, public contracts, operations, or user experience.

## Workflow
Use `docs/skillset/common-skill-workflow.md` for the common define/plan/build/verify/review/handoff loop. Keep this skill focused on: Collect files changed, commands run, validation results, risks, and next steps before completion.

## Evidence required
- Files read and changed.
- Commands run and outcomes.
- Test or validation coverage.
- Known risks and explicit exclusions.

## Handoff
- Summarize changes, evidence, risks, and next steps.
