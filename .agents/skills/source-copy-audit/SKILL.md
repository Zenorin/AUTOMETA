# Source Copy Audit

## Use when
- The task matches this skill's description.
- The change may affect durable behavior, tests, docs, public contracts, operations, or user experience.

Use `docs/skillset/common-skill-workflow.md` for the common define/plan/build/verify/review/handoff loop. Keep this skill focused on: Check that generated implementation does not copy restricted source text/assets/identifiers.

- Commands run and outcomes.
- Test or validation coverage.
- Known risks and explicit exclusions.

## Handoff
