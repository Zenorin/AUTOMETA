# Pass Manifest Verification

## Use when
- The task matches this skill's description.
- The change may affect durable behavior, tests, docs, public contracts, operations, or user experience.

Use `docs/skillset/common-skill-workflow.md` for the common define/plan/build/verify/review/handoff loop. Keep this skill focused on: Verify staged pass/manifests and evidence before declaring phase completion.

- Test or validation coverage.
- Known risks and explicit exclusions.

## Handoff
